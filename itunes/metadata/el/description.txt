Δημοφιλές παιχνίδι Gold Miner έρχεται να GOOGLEPLAY με πολυτελή γραφικά.

Εθιστικό gameplay, όμορφα γραφικά και φανταστική μουσική σίγουρα ικανοποιεί το παιχνίδι σας.

Απεριόριστα επίπεδα, εντελώς δωρεάν και έτσι δεν θα είναι σε θέση να βάλει κάτω Gold Miner Deluxe.
 
Είσαι ένας ανθρακωρύχος χρυσό και είναι η εργασία σας για να εξορύξουν όσο χρυσό, διαμάντια και να βγει με το πιο ποσό των χρημάτων που μπορείτε. Πέτρα, ποντίκι, TNT αν μπορούν να είναι τα εμπόδια για να σας κάνει να αργή. Θα πρέπει να αγώνα με το χρόνο για να ολοκληρώσει το έργο σε τόπο και στη συνέχεια να πάνε σε άλλη μέρη. Στο παιχνίδι θα είναι σε θέση να ταξιδέψει σε πολλά όμορφα μέρη για να ορυχείου, όπως δάσος, σπηλιά, βουνό, θάλασσα και την πόλη.

Μετά την ολοκλήρωση ενός επιπέδου, θα μπορούσε να είναι εξοπλισμένα με τη δύναμη-up στοιχεία όπως το δυναμίτη, δύναμη, τυχερό λουλούδι, το βιβλίο συλλογή πετρωμάτων, Πολωνικά διαμάντι. Θα σας βοηθήσει να εύκολα να παίξει στο επόμενο επίπεδο. Εξερευνήστε το παιχνίδι να γνωρίζουν πώς να χρησιμοποιούν κάθε στοιχείο. Θα πρέπει να δαπανήσει χρήματα που κερδίζουν για να αγοράσει τα στοιχεία, οπότε η συμβουλή μας είναι να είναι έξυπνος για να επιλέξουν να αγοράσουν τα αντικείμενα.

Gold Miner Deluxe θα ξεκινήσει με την κλασική σχεδίαση επίπεδο και το περιβάλλον που σας κάνουν να νιώσετε οικεία. Μπορείτε επίσης να συγκρίνετε το σκορ σας με τους φίλους ή άγνωστους εχθρούς όλο τον κόσμο με PlayGame leaderboard
